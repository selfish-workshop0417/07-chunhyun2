# 오늘의 자리 (Today's Seat)

우리 회사 직원들의 휴가, 조퇴, 재택근무, 해피데이(오전/오후)를 한눈에 볼 수 있는 사내 근태 현황 대시보드.

## 프로젝트 정보
- **유형**: 대시보드
- **타겟 사용자**: 전 직원
- **출시 목표**: 2026년 6월

## 핵심 기능
- 오늘 부재자 한눈에 보기
- 월간 캘린더 뷰

## 폴더 구조 (5개 팀)
- `planning/` — 기획 팀: 무엇을 만들 것인가
- `design/` — 디자인 팀: 어떻게 보여줄 것인가
- `cx/` — CX 팀: 누가 어떻게 사용할 것인가
- `marketing/` — 마케팅 팀: 어떻게 알릴 것인가
- `strategy/` — 전략 팀: 어떻게 지속 가능하게 만들 것인가
